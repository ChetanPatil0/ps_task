## Getting Started

set the env variables to the root

.env.local

GEMINI_API_KEY=
QDRANT_URL=
QDRANT_API_KEY=
MYSQL_HOST=
MYSQL_USER=root
MYSQL_PASSWORD=
MYSQL_DATABASE=ps_task

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
